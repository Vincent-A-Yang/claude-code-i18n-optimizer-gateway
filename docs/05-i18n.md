# 05. Claude Desktop 汉化

## 目标

汉化 Claude Desktop 中未覆盖的开发者设置、MCP、extensions、plugins、Claude Code、gateway 相关界面文案。

## 安全原则

- 只替换完整字符串字面量。
- 不替换变量名、函数名、协议字段、class 名或组件名。
- 每轮只处理一组页面。
- 修改前备份目标文件。
- 修改后执行 `node --check`。
- 失败立即恢复备份。

## 推荐流程

1. 找到当前 Claude Desktop app 根目录。
2. 找到包含目标英文文案的 JS chunk。
3. 备份将要修改的 JS 文件。
4. 使用翻译表 dry-run：

```powershell
.\scripts\Patch-ClaudeI18n.ps1 `
  -ClaudeAppRoot "C:\Path\To\ClaudeApp" `
  -TranslationFile .\translations\zh-CN.sample.json `
  -DryRun
```

5. 确认命中字符串合理后再应用。
6. 对每个修改过的 JS 文件执行 `node --check`。
7. 重启 Claude。
8. 检查目标页面和日志。

## 什么不该汉化

- 模型名称。
- provider 名称。
- API 路径。
- HTTP header。
- JSON key。
- 插件包内部要求保留的英文标识。

## 常见残留英文

残留英文通常来自：

- 新加载 chunk。
- 远端数据。
- 插件或扩展自己的包。
- 模型、协议或 provider 名称。

优先通过搜索完整英文文案补充翻译表，不要扩大到模糊替换。
